Projecte de Programació - Subgrup PROP 41.2
Sistema de gestió i anàlisi d'enquestes amb clustering (K-Means i K-Medoids)

Estructura del projecte:
- FONTS/: Codi font Java, tests, datasets i configuració de build
- DOCS/: Documentació del projecte i diagrames UML
- EXE/: Fitxers executables compilats (.class) generats per Make/Gradle
- Alumnes.txt: Informació dels membres del grup
- ReadMe.md: Instruccions d'ús (Makefile i Gradle)

Per començar:
1. Aneu al directori FONTS/
2. Compileu amb Make: make all
3. Executeu amb Make: make run
4. Tests amb Make: make test

Alternativament amb Gradle:
- Compilar i tests: ./gradlew build test
- Executar (si s'escau): ./gradlew run

Vegeu ReadMe.md per més detalls sobre les comandes disponibles.
